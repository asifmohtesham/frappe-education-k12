# Phase 0 — Foundation Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Stand up the complete development foundation for the K-12 SaaS: bench environment, the `education_k12` app scaffolded inside this monorepo, the upstream dependency question answered, a proven test loop (local + CI), the portal SPA shell with auth and Arabic/RTL plumbing, and a first-cut school provisioning script.

**Architecture:** Custom Frappe app (`education_k12`) layered on unmodified upstream `frappe/education` (+ ERPNext if required), developed in a WSL2 bench, living in this monorepo at `apps/education_k12` and symlinked into the bench. A Vue 3 + Frappe UI SPA is embedded in the app and served at `/portal`. Ops scripts live in `ops/` as importable, unit-tested functions wrapped by a CLI.

**Tech Stack:** Frappe Framework v15, ERPNext v15 (pending Task 3 verification), frappe/education v15, Python 3.11, MariaDB 10.6, Vue 3, Vite, frappe-ui, vue-i18n, vitest, pytest, GitHub Actions.

**Spec:** `docs/superpowers/specs/2026-06-12-k12-education-saas-design.md`

`src/data/session.js`:

```js
import { computed, reactive } from 'vue'
import { createResource } from 'frappe-ui'

export function sessionUser() {
  const cookies = new URLSearchParams(document.cookie.split('; ').join('&'))
  let user = cookies.get('user_id')
  if (user === 'Guest') user = null
  return user
}

export const session = reactive({
  user: sessionUser(),
  isLoggedIn: computed(() => !!session.user),
  login: createResource({
    url: 'login',
    makeParams({ email, password }) {
      return { usr: email, pwd: password }
    },
    onSuccess() {
      session.user = sessionUser()
      window.location.replace('/portal')
    },
  }),
  logout: createResource({
    url: 'logout',
    onSuccess() {
      session.user = null
      window.location.replace('/portal/login')
    },
  }),
})
```

```python
"""Provision a new school site on the bench.

First cut (Phase 0): create site, install apps, set the school display name.
Designed as importable functions so a future control-plane app can call
`provision()` directly; the CLI is a thin wrapper.

Run from the bench directory:
    python3 /path/to/ops/provision_school.py \
        --site alnoor.localhost --school-name "Al Noor International School" \
        --admin-password ... --db-root-password ...
"""

import argparse
import subprocess
from dataclasses import dataclass

# Adjust per docs/decisions/0001-erpnext-dependency.md
APPS = ("erpnext", "education", "education_k12")


@dataclass
class SchoolConfig:
    site_name: str
    school_name: str
    admin_password: str
    db_root_password: str
    default_language: str = "en"


def build_commands(cfg: SchoolConfig) -> list[list[str]]:
    """Return the ordered bench commands that provision a school site."""
    site = ["--site", cfg.site_name]
    cmds = [
        [
            "bench", "new-site", cfg.site_name,
            "--admin-password", cfg.admin_password,
            "--db-root-password", cfg.db_root_password,
        ]
    ]
    cmds += [["bench", *site, "install-app", app] for app in APPS]
    cmds.append(
        ["bench", *site, "set-config", "lang", cfg.default_language]
    )
    cmds.append(
        [
            "bench", *site, "set-value",
            "K12 Settings", "K12 Settings",
            "school_display_name", cfg.school_name,
        ]
    )
    return cmds


def provision(cfg: SchoolConfig, runner=subprocess.run) -> None:
    """Execute the provisioning commands; fails fast on the first error."""
    for cmd in build_commands(cfg):
        runner(cmd, check=True)


def main() -> None:
    parser = argparse.ArgumentParser(description=__doc__)
    parser.add_argument("--site", required=True, dest="site_name")
    parser.add_argument("--school-name", required=True, dest="school_name")
    parser.add_argument("--admin-password", required=True)
    parser.add_argument("--db-root-password", required=True)
    parser.add_argument("--default-language", default="en", choices=["en", "ar"])
    args = parser.parse_args()
    provision(SchoolConfig(**vars(args)))


if __name__ == "__main__":
    main()
```
